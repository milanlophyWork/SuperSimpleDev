How to use external libraries and javascript modules together?

We use ESM version of the library for this purpose. ESM stands for EcmaScript Module. EcmaScript is just an another name for JavaScript. It is a version that works with JavaScript modules.

Note : Not every library has esm version. Hence for some we need to use script tag instead of importing.

Default Export

Default export is an another way of exporting something from a file. We can use it when we want to export only 1 thing.
