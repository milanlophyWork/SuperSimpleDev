Functions

A function lets us reuse code. syntax : function fnName(parameter _list) { fnBody } . calling the function : fnName(). Functions also make our code easier to update.

Return statement

Return statement let us to get a value out of a function. When we use a return statement , it ends the function immediately. Code after return won't work. If we return without a value (eg: return ;) then undefined is returned.

Parameters

Parameters put a value into a function. Parameters work the same way as a variable. we can save a value inside parameter and use it as a variable. Parameters only works inside a fn. We can have many paramters in a fn.
