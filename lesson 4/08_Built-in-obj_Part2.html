Objects provided by the language are built-in objects. console.log and Math.random are built-in objects. Other built-in objects are JSON and localStorage.

JSON built-in obj helps us work with JSON (JavaScript Object Notation). JSON is basically a syntax similar to JavaScript object but it has less features.

JSON does't support single quotes and fns like Js. All strings in JSON must use double quotes. Then why use JSON over JS ?

JS object only make sense in Js. JSON is understandable by all programming languages. ie JSON syntax is more universal. Hence we use JSON when we send data between computers and when we store data.

JSON object helps us convert a JavaScript object to JSON.

localStorage built-in object save values more permanently. Variables are temporary. If we refresh/close the page, all variables are deleted. localStorage stores values even if we refresh or close the page.

localStorage only support strings. To store values => localStorage.setItem('name', 'value to be stored'). To get stored values => localStorage.getItem('name')
