Dom is another built in object called document object. document.title can be used to access the title of webpage and also to change the webpage. document.body gives us access to html body element. ie we can have html elements inside JS. DOM commbines Js and html together.

When an html element is inside Js , the html element is converted to Js object. innerhtml property in body object gives access to all html inside body. innerHTML can be replaced with new html code instead of text.

DOM methods

document.querySelector() lets us get any element from the page and put it inside Js. Every html element in Js (except input element because input elements have no html inside) has a property called innerHTML to access the html inside that element. querySelector only access first matching element.

To get other elements we use class attribute. class attribute adds a label to the elements. Use dot at starting for access.
