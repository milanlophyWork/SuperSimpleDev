Array Methods

forEach() method :

We loop through an array using for loop or a while loop. Another way is by using the method forEach() which is the preferred way to loop through an array. Because this is easier to read than a for loop and while loop.

forEach() has a feature : the function can have 2 parameters , value and index (gives position of value in the array)

filter() method :

This method creates a new array. filter((value, index)=> {}) accepts a function that returns a value (true or false). Values satisfying the condition (ie values that are truthy or return true) are inserted into the array.

It let us filter out values based on a criteria. So called filter()

map() method :

Used to transform an array into another array.

Arrow Functions : Shorter way to write functions

Closure : If a function have a access to a value, it will always have access to that value.
