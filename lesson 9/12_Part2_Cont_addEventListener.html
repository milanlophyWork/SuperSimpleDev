addEventListener

Every html element has a method called addEventListener(). It lets us run some code when we interact with the element. addEventListener() has 2 parameters :

1. event => What type of interaction we want the element to be listened for

2. function => To be runned when event occurs

Advantages of addEventListener() over attributes.

* It lets us add multiple event listeners for an event

* we can remove an event listener using a method called removeEventListener(event, fn we want to remove)

In conclusion, addEventListeners gives us more control over eventListeners. Hence It is better to use eventListeners instead of attributes like onclick.
